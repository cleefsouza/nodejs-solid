# :seedling: SOLID com Node.js :seedling:

> Princípios SOLID em uma API REST com Node.js e TypeScript - Rocketseat.

### Requisitos
- Node
- NPM/YARN

### Executando
- Clone o projeto:
  ```shell
  git@github.com:cleefsouza/nodejs-solid.git
  ```

- Execute o comando na pasta raiz:
  ```shell
  npm install
  # ou
  yarn install
  ```

- Rode o projeto:
  ```shell
  npm run start
  # ou
  yarn start
  ```
